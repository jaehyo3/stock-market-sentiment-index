# 📈 모아주식 (Moa Stock)

### AI 댓글 감성분석 점수를 활용한 국내 주식시장 심리지수 서비스

'모아주식'은 AI를 활용해 온라인 주식 토론방의 댓글 감성을 실시간으로 분석하고, 이를 지수화하여 복잡한 시장 심리를 직관적으로 파악할 수 있도록 돕는 웹 서비스입니다. 투자자들이 객관적인 데이터에 기반한 합리적인 의사결정을 내릴 수 있도록 지원하는 것을 목표로 합니다.

## ✨ 주요 기능

*   **실시간 시장 심리 지수 제공**: AI 감성분석 모델이 분석한 댓글 데이터를 기반으로 현재 주식 시장의 전반적인 분위기(긍정/부정/중립)를 점수로 시각화하여 제공합니다.
*   **종목별 상세 분석**: 특정 종목을 검색하여 해당 종목에 대한 최신 뉴스, 재무 정보, 그리고 AI가 분석한 댓글 심리 점수 추이를 종합적으로 확인할 수 있습니다.
*   **관심 종목 코멘트 및 평가 확인**: 사용자가 관심 있는 종목의 최신 댓글들과 긍정/부정 평가를 필터링하여 여론의 흐름을 쉽게 파악할 수 있습니다.

## 🙋‍♂️ 제가 담당한 역할 (신재효)

저는 본 프로젝트에서 **백엔드, 프론트엔드 개발 및 데이터베이스 관리**를 담당하며 사용자가 직접 마주하는 서비스의 핵심 기능들을 구현했습니다. 1차 프로젝트의 경험을 바탕으로 사용자 피드백을 적극 반영하여 서비스 완성도를 높이는 데 기여했습니다.

### 🖥️ Frontend
*   **메인 페이지 UI 개발 및 미니그래프 구현**:
    *   웹사이트의 첫인상을 결정하는 메인 페이지의 전체적인 레이아웃을 구성했습니다.
    *   AI 모델이 분석한 시장 심리 지수 데이터를 API로 받아, 이를 시각적으로 표현하는 미니그래프를 JavaScript를 사용하여 동적으로 구현했습니다.
*   **사용자 인증 관련 페이지 개발**:
    *   로그인, 회원가입, 회원정보 수정 등 사용자 인증에 필요한 모든 페이지의 UI를 개발하고, 유효성 검사 로직을 적용했습니다.

### ⚙️ Backend
*   **사용자 관리 API 개발**:
    *   회원가입, 로그인, 로그아웃, 회원정보 수정 등 사용자 관리의 핵심 로직을 Python을 사용하여 RESTful API 형태로 구현했습니다.
*   **이메일 인증 시스템 구축**:
    *   보안 강화를 위해 회원가입 시 사용자의 이메일로 인증 코드를 발송하고, 이를 확인하는 백엔드 로직을 구현했습니다.

### 🗃️ Database
*   **테이블 설계 및 관리**:
    *   사용자 정보, 관심 종목 등을 효율적으로 관리하기 위한 데이터베이스 스키마를 설계하고 SQL을 통해 테이블을 생성 및 관리했습니다.

## 🌱 성장 및 회고 (Challenges & Learnings)

*   **기획과 구현의 간극을 줄이는 경험**: 1차 프로젝트를 통해 아이디어를 실제 구현 가능한 기능으로 구체화하는 과정의 중요성을 배웠습니다. 2차 프로젝트에서는 팀원들과의 적극적인 소통을 통해 실현 가능성에 초점을 맞추어 기획 단계부터 참여하며 완성도 높은 결과물을 만들 수 있었습니다.
*   **팀원 간의 적극적인 지식 공유의 가치**: 프로젝트 초반에는 각자의 역할에만 집중하여 다른 팀원의 작업 진행 상황을 깊이 이해하지 못하는 어려움이 있었습니다. 이를 해결하기 위해 정기적인 스크럼을 통해 서로의 진행 상황과 기술적 이슈를 공유했습니다. 그 결과, 프론트엔드와 백엔드, 그리고 AI 모델 간의 데이터 연동을 더욱 원활하게 진행할 수 있었고, 협업의 중요성을 몸소 깨닫는 계기가 되었습니다.
